iOS开发之-UnitTests、UITests

这是iOS开发的一大神器。能快速的测试自己的项目，而且不要去麻烦测试MM就能自己搞定测试环节，特别是公司的测试很忙的时候，那你更加要学习了。
本教程只是一个简单的教程，任何东西都是从简单开始。我们就不从Hello word开始了。这个也太…
从每个App都有的登陆注册开始。我最喜欢写登陆注册了😄。
Demo地址：https://github.com/liss1990/UITestDemo

##1、新建一个工程
在新建的时候注意够上两个选项:

  • Include Unite Tests —–//这个是表示单元测试
  • Include UI Tests —–//这个代表UI测试 可以让你的执行的步骤重新执行。你的UI操作的脚本都写在这里

如果没有勾上，后面你也可以在 Targets 里面去新建。后面新增如图:

###先介绍几个方法

  • -(void)setUp ; ////这是开始
  • 在setup 里面有这个方法 [[[XCUIApplication alloc] init] launch]; 这是启动程序
  • -(void)tearDown;///这里是结束 释放资源
  • -(void)testExample;///这里就是你执行脚本的方法。

这里有一些我们之前没用过的东西
XCUIElement：是继承NSObject，实现协议XCUIElementAttributes, XCUIElementTypeQueryProvider用来表示系统的各种UI元素。
